# PingMonitorPy
Aplicação direcionada ao monitoramento da conexão da internet, com interface e criação de .CSV para armazenamento das informações.

Python 3 necessário para a execução 

Extensão nencessária para execução
- python -m pip install pyinstaller

Gerar o .exe
-python -m PyInstaller --onefile --windowed --name "MonitorConexao" ping_monitor.py
